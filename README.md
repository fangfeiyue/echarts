# echarts-demo
在大数据与云计算的时代，数据的获取、存储、搜索、共享、分析以及可视化的呈现，是当前很重要的研究方向。Echarts是国人开发的基于canvas的可视化库，以api设计优雅，效果酷炫等优点，越来越受广大前端开发者喜爱，本项目要介绍echarts的入门使用，并配套详细的代码，后面通过实战小项目再次加深对echarts的理解，并且掌握echarts在实战中的用法。
